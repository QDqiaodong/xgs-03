
# 植物养护项目需求20道

## 项目简介

PlantCare 是一款面向植物爱好者的全栈植物养护管理应用，前端基于 Vue 3 + Vite 构建，后端采用 Spring Boot + MySQL + Redis 技术栈，通过 Docker Compose 实现容器化部署。项目核心模块包括：植物品种百科库（提供光照、浇水、温度、施肥等养护知识）、我的植物档案管理（记录每株植物的自定义信息与养护周期）、养护日志系统（浇水/施肥/修剪/换盆等操作记录与长势追踪）、交流社区（求助问答与经验分享，支持评论、点赞、最佳答案）、收藏功能等，致力于帮助用户科学养护植物、交流养护经验。
